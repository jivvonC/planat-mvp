# Project Overview

## Project Name

Planat

---

# Project Goal

Planat is a mobile-first plant care application designed for beginner plant owners.

The goal of this project is **not to build a production-ready service**, but to create a realistic interactive prototype for usability testing.

The prototype should feel polished enough that users naturally interact with it during interviews while allowing rapid iteration based on UX findings.

---

# Vision

People don't simply own plants.

They build relationships with them.

Planat encourages users to care for plants through small daily interactions rather than treating plant care as a list of maintenance tasks.

Inspired by _The Little Prince_, every plant becomes "my own special plant" because of the time and attention invested in it.

---

# Design Philosophy

The application should feel:

- Warm
- Calm
- Emotional
- Friendly
- Gentle
- Minimal
- Soft

Avoid making the interface feel technical, clinical or data-heavy.

The product should evoke the feeling of taking care of a tiny planet.

---

# Product Experience

The application should encourage users to

- observe
- care
- discover
- build habits
- feel emotionally attached

instead of simply checking off tasks.

Every interaction should reinforce the relationship between the user and their plants.

---

# Target Users

Primary users:

• Beginner plant owners

• People living alone

• Users who have accidentally killed plants before

• Users who want guidance rather than expert knowledge

---
